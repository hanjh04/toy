### git bash 사용
1. install git from `https://git-scm.com/download/win`
2. `ctrl` + `\`` 로 터미널 열기
3. `ctrl` + `shift` + `p` 로 command palete열기
4. `Terminal: select default shell` 검색 후 선택
5. `Git Bash` 선택
6. 재시작
